Conheça o bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

Containers - O que são Containers

Containers são os elementos de layout mais básico do Bootstrap e são necessários quando usamos o sistema de grid padrão. Os contêineres são o elemento de layout mais básico do Bootstrap e são necessários ao usar nosso sistema de grade padrão . Os contêineres são usados ​​para conter, preencher e (às vezes) centralizar o conteúdo dentro deles. Embora os contêineres possam ser aninhados, a maioria dos layouts não exige um contêiner aninhado.
